# MuiltiResourceTerraform
Projeto criado para desenvolver/praticar estudos feitos em cima do terraform utlizando a AWS como provider
## Requisitos
- Conta na AWS
- Criação de um S3 Bucket
- Gerar sshkey
- Terraform Instalado
## Como utilizar
- Pull dos arquivos .tf
- terraform init
- terraform validate
- terraform fmt
- terraform apply
- terraform destroy
